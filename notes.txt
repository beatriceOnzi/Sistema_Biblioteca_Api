3. Como colocar para funcionar
Abra o terminal na pasta projeto_estudo.

Instale as dependências executando: pip install -r requirements.txt

Rode o servidor executando: python app.py

Abra o seu navegador e acesse: http://127.0.0.1:5000

💡 Resumo do Fluxo de Comunicação (Para seus estudos)
Ação: Você digita algo e clica em "Adicionar" no HTML.

JS (Front): O main.js pega esse texto, empacota num formato chamado JSON e envia uma requisição POST para o endereço /api/tarefas do seu servidor.

Flask (Back): O app.py percebe que alguém chamou a rota /api/tarefas via POST. Ele desempacota o JSON.

SQLAlchemy (Banco): O Flask usa o SQLAlchemy para traduzir o texto em um comando SQL (INSERT INTO...) e salva a tarefa no arquivo database.db.

Retorno: O Flask envia um "Ok" de volta para o JavaScript. O JS recebe o Ok e pede imediatamente a lista atualizada para o servidor exibir na tela.
